https://coveralls.io/github/kamilkisiela/angular2-now

**Travis-CI:**

https://travis-ci.org/kamilkisiela/angular2-now

How to use?
-----------

**Install**

> npm i

**Local testing**

On chrome with autowatch and disabled singleRun

> npm run test:local

**CI testing**

On firefox with disabled autowatch and enabled singleRun

> npm run test
